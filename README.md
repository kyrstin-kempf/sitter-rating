# SitterRating: React/Rails API

## Description

SitterRating allows a user to signup, login, create new sitters, and create, edit, and delete ratings. On login, the user will see the sitters they have rated when the option to navigate to view all sitters.

## Requirements

To run this application, make sure the following is installed:
- ruby 2.7.4
- node 16.16
- npm version 9.4

## How To Use

1. Fork and clone this repository
2. Run `bundle install`
3. Run `rails db:migrate db:seed`
4. Run `rails s` to start the backend on [http://localhost:3000](http://localhost:3000)
5. In another terminal, navigate to client directory `cd client`, then run `npm start` to start the frontend on [http://localhost:4000](http://localhost:4000).

To view SitterRating without forking and cloning, visit [https://sitter-rating.com](https://sitter-rating.onrender.com)
